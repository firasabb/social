# Social

A mini app built with Laravel.


## Usage

- First of all, rename .env.example to .env
- Modify the following fields in .env file to fit your environment:

```env

APP_URL=

DB_CONNECTION=mysql
DB_HOST=
DB_PORT=
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=

FACEBOOK_CLIENT_ID=
FACEBOOK_CLIENT_SECRET=

INSTAGRAM_KEY= 
INSTAGRAM_SECRET=
```
- Migrate the database.
- Run:
```bash
npm install
npm run dev
```
